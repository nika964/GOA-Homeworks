HTML- არის საიტის ჩონჩხი(სტრუქტურა) სადაც შეგვიძლია შევქმნათ ღილაკები, ტექსტები, ჩავსვათ ფოტოები ანუ კონტენტი.

HTML- is a framework (structure) where we can create buttons, texts, insert photos or content.

CSS- არის გამსტილავი ხელსაწყო რომელიც ჩვენს HTML ელემემტებს აძლევს სტილს პოზიციებს და ასე შემდეგ ანუ საიტი ხდება უფრო ლამაზი და ვიზუალურად დახვეწილი.

CSS- is a styling tool that gives our HTML elements style positions and so on, so the site becomes more beautiful and visually refined.

JAVASCRIPT- არის როგორც ფუნქციონალი ანუ ჩვენ რო რაღაცას დავაჭიროთ და მოხდეს რაღაცა ამისთის საჭიროა შეიქმნას ფუნქციები.

JAVASCRIPT is like functionality, that is, if we press something and something happens, we need to create functions.
